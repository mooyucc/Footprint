## 数据结构说明

每个国家在代码中包含以下信息：

- **ISO 国家代码**：枚举的 `rawValue`（如 "CN"、"US" 等）
- **中文名称**：`displayName` 属性返回的中文名称
- **英文名称**：`englishName` 属性返回的英文名称
- **国旗 emoji**：`flag` 属性返回的国旗表情符号

## 使用方法

在代码中访问国家列表：

```swift
// 获取所有国家
let allCountries = CountryManager.Country.allCases

// 获取某个国家的中文名称
let countryName = CountryManager.Country.china.displayName  // "中国"

// 获取某个国家的英文名称
let englishName = CountryManager.Country.china.englishName  // "China"

// 获取某个国家的国旗
let flag = CountryManager.Country.china.flag  // "🇨🇳"
```
